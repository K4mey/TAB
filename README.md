## Compiling
Plugin can be compiled using maven.

```mvn package```

## Maven Repository
```
<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>

<dependencies>
  <!-- TAB -->
  <dependency>
    <groupId>com.github.NEZNAMY</groupId>
    <artifactId>TAB</artifactId>
    <version>2.9.0</version>
  </dependency>
</dependencies>
 ```
